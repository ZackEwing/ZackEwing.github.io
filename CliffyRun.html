Project Scope

Cliffy Run is the first mobile release from Bad Cloud Games. It is an addictive endless runner with dozens of quirky characters and action-packed environments.

Contribution

Cliffy Run was developed by a two man team. My role as artist included generating all visual assets, 3D models, particle effects and animations for the game world, the UI, the Bad Cloud Games website, and all app store and marketing assets.
